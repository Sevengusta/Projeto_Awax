# Projeto_Awax

Este projeto de template de site foi desenvolvido durante as aulas do curso que estou fazendo. Foi utilizado os conceitos básicos de HTML e CSS, com o destaque do uso do CSS FlexBox.

Foi utilizado também os conceitos de responsividade com o uso do recurso do CSS @media e com o uso de eventos básicos em JavaScript para o menu funcionar de forma responsiva.
